# 日历模块重构计划

**问题现状**:
1. Dashboard 没有日历卡片展示
2. 无法新建日程（无创建入口）
3. 只有列表视图，无日历视图
4. 与任务/流程无关联

**产品定位**:
- 个人/团队的日程管理中心
- 与任务系统联动（任务截止日期自动同步为日程）
- 支持手动创建日程（会议、截止、评审等）

---

## 功能规划

### Phase 1: 基础功能（本次实现）
- [x] API 支持创建日程（POST /api/events）
- [ ] Dashboard 显示日历卡片（近期日程）
- [ ] 日历模块单页视图优化
- [ ] 新建日程按钮和浮窗

### Phase 2: 日历视图
- [ ] 月历视图（显示当月所有日程）
- [ ] 周历视图
- [ ] 拖拽创建日程

### Phase 3: 任务联动
- [ ] 任务截止日期自动同步到日历
- [ ] 日程可关联任务
- [ ] 日程提醒功能

---

## 本次实现（Phase 1）

### 1. Dashboard 日历卡片
显示近期 3-5 个日程，点击"查看全部"进入单模块视图

### 2. 新建日程浮窗
- 标题（必填）
- 开始时间（必填）
- 结束时间（可选）
- 类型（会议/截止/评审/其他）
- 备注（可选）

### 3. 日历列表优化
- 空状态显示"新建日程"按钮
- 筛选器优化（类型、时间范围）
- 支持删除日程

---

## 技术实现

### API
- `GET /api/events` - 获取日程列表（已有）
- `POST /api/events` - 创建日程（已有）
- `PATCH /api/events` - 更新日程（待实现）
- `DELETE /api/events` - 删除日程（待实现）

### 组件
- `CalendarList` - 日历列表（已有）
- `CreateEventModal` - 新建日程浮窗（新增）
- `CalendarCard` - Dashboard 卡片（新增）
