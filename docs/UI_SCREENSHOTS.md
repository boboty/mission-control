## 🎨 设计说明

### 颜色方案
- 主色：`var(--color-primary)` - 蓝色系
- 成功：`var(--badge-success-bg)` - 绿色
- 警告：`var(--badge-warning-bg)` - 黄色
- 错误：`var(--badge-error-bg)` - 红色

### 交互细节
1. **标签页切换**: 平滑过渡，底部边框高亮
2. **时间线**: 圆点 + 连线（视觉引导）
3. **关联对象**: 卡片式，hover 效果
4. **操作按钮**: Primary/Secondary/Ghost 变体

### 响应式
- 最大宽度：`max-w-2xl` (672px)
- 最大高度：`max-h-[90vh]`
- 内容区可滚动：`overflow-y-auto`
